## 🎯 Project Overview

This project builds an end-to-end **AI Intelligence Dashboard** for digital marketing analytics, combining:

- 📊 **Exploratory Data Analysis** across 500+ campaign records
- 🤖 **Machine Learning** models to predict ROAS and detect anomalies
- 💡 **AI-generated insights** using LLMs (OpenAI API)
- 📈 **Interactive visualizations** with Plotly and Streamlit
